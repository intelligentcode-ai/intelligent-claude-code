# Activate Role

Switch to specialized role and adapt behavioral context.

## Usage
```bash
icc-activate-role @Role-Name [--minimal] [--chain]
```

## Behavior
- Load role expertise and constraints
- Switch communication style and focus
- Create dynamic specialists if needed
- Preserve role scores and context

## Examples
```bash
icc-activate-role @PM              # Coordinate and delegate
icc-activate-role @AI-Engineer     # Technical AI/ML focus
icc-activate-role @React-Developer # Auto-create React specialist
```

## Dynamic Creation
Creates @Domain-BaseRole specialists with relevant expertise when needed.
